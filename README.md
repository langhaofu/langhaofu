### 基础控制与电子设计项目

**1. 智能烟雾与一氧化碳检测报警器 (独立开发)**
* **技术栈**：STM32 + 气体传感器 + ESP8266 + Qt 上位机
* **内容**：实现对环境烟雾浓度及 CO 含量的实时监测。独立设计电路及程序，并编写跨平台上位机软件，支持数据的远程曲线显示、阈值自定义设置及异常语音报警。

**2. 基于树莓派/ESP32 的嵌入式智能视觉追踪终端 (独立开发)**
* **内容**：部署人脸识别与目标检测模型，设计主从机无线链路，联动底层单片机控制云台进行高精度 PID 跟随。获“华为杯”嵌入式设计竞赛省**二等奖**、电赛省**三等奖**。

**3. 迷宫自动寻路小车 (独立开发)**
* **内容**：结合 STM32 与 OpenMV，实现环境感知与空间建图，编写路径规划算法使小车在 30 秒内自主规避障碍并驶出迷宫。

**4. 高精度电磁曲射炮系统 (负责程序设计)**
* **内容**：负责硬件搭建与发射系统。采用 STM32 自动解算弹道参数，精准调整炮管方向，确保弹丸命中 2-3 米内环形靶。
